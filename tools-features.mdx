## Current Tools

### store_session

Save AI conversation sessions as organized markdown files.

**Features:**
- Auto-organization by date (`.codearchitect/sessions/YYYY-MM-DD/`)
- Smart topic extraction
- Zero configuration (auto-detects project root)

**Usage:**
```
Store this conversation about implementing authentication
```

See [API Reference](/api-reference/store-session) for details.

### get_session

Retrieve stored AI conversation sessions with TOON format support.

**Features:**
- Get specific session by filename
- List all sessions (optionally filtered by date)
- **~40% token reduction** with TOON format
- Automatic format selection (`auto` mode)
- Support for limiting results

**Usage:**
```
Get session session-20250115-143022-authentication-implementation.md
```

```
List all sessions from 2025-01-15
```

See [API Reference](/api-reference/get-session) for details.
