## Format

All tools follow MCP standard:
- **Request**: JSON-RPC 2.0
- **Response**: Structured JSON with success/error
- **Auth**: None required (local only)

## Response Format

**Success:**
```json
{
  "success": true,
  "data": { ... },
  "message": "Operation completed"
}
```

**Error:**
```json
{
  "success": false,
  "error": "ERROR_CODE",
  "message": "Error description"
}
```
